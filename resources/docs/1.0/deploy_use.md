## Deploy z deva

Deployment z aktualnego deva jest możliwy w każdej chwili, aby tego dokonać należy:
- przejść do [kontenera dockerowego]({{route}}/{{version}}/docker_use#section-2)
- użyć komendy 
```powershell
php artisan deploy production
```
